# matlab-telecoms-projects
## 🛠️ Main Projects & Simulations

### 1. Behavioral Modeling of Single-Photon Avalanche Diodes (SPADs)
* **Directory:** `/spad-behavioral-modeling`
* **Description:** Developed a comprehensive simulation framework to model the full quenching and recovery cycle of a SPAD (specifically calibrated for the PGA-314-50 detector).
* **Key Achievements:**
  * Adjusted and calibrated circuit electrical parameters ($R_q$ and $C_{total}$) based on manufacturer datasheets.
  * Successfully modeled time constants and avalanche behavior under different operating modes.
  * Validated experimental simulation curves against theoretical optical detection constraints.

### 2. Wireless Communication Channel & Signal Propagation
* **Directory:** `/wireless-channel-simulation`
* **Description:** Performance analysis and simulation of wireless telecommunication links, modeling noise, interference, and bit error rate (BER).
* **Key Achievements:**
  * Implemented signal propagation models to analyze channel capacity under varying conditions.
  * Generated precise graphical analyses to support secure transmission architecture decisions.
